# IG-Planner

Plan your IG grid on local html

![example](/example.jpg)

## How to use
Write a path for image like below:

```
~~~

<body style="background-color: black;">
    <div class="ig-grid">
        <img src="/your/dir/pic4.jpg">
        <img src="/your/dir/pic3.jpg">
        <img src="/your/dir/pic2.jpg">
        <img src="/your/dir/pic1.jpg">
    </div>
</body>

</html>
```
